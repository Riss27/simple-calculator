# Simple Calculator

A basic web calculator built with HTML, CSS, and JavaScript.

## 🚀 Features

- Basic arithmetic operations: +, −, ×, ÷
- Clear (AC) button
- Responsive layout with clean UI

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
